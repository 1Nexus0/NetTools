# Contributing to Nexus.NetTools.IPRangeParser

Thank you for considering contributing!

## How to Contribute

1. **Create a feature branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```
2. Make your changes
3. Run tests
   ```bash
   dotnet test
   ```
4. Commit your changes
   ```bash
   git commit -m "feat: add your feature description"
   ```
5. Push to your branch
   ```bash
   git push origin feature/your-feature-name
   ```
6. Create a Pull Request
   
## Pull Request Guidelines

  - One PR per feature/bugfix - Keep changes focused

  - Write clear commit messages - Use conventional commits

  - Update documentation - Update README.md if needed

  - Add tests - Cover new functionality

  - All tests must pass - CI will verify
## Development Setup
```bash
# Clone
git clone https://github.com/your-username/Nexus.NetTools.IPRangeParser.git

# Build
dotnet build

# Run tests
dotnet test
```
